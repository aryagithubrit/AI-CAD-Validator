# AI CAD Validation System

##  Overview
This project is an AI-assisted CAD validation system that helps engineers detect design errors in early stages.

##  Features
- Real-time design validation
- Error and warning detection
- Intelligent scoring system
- Design improvement suggestions

##  Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)

##  Use Case
Helps reduce design errors, cost, and development time in engineering workflows.
